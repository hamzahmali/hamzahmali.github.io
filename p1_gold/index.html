The Legend of Zelda – First Dungeon

By Hamzah Ali

Partners: Joshua Ma, Vera Sivrieva

This project is a recreation of the first dungeon of the classic Nintendo game The Legend of Zelda. The goal was to recreate the feel and structure of the NES game using Unity and C#.

My Contributions

Health, Damage, Knockback, and Hit Flash

I implemented the health and damage system for both the player and enemies. I added knockback and a visual hit flash effect to improve combat feedback.

Room Transitions

I created the room transition system that moves the player between rooms through doors, matching the flow of the original Zelda game.

Boomerang

I built the boomerang weapon, including throwing, returning, and interaction logic with enemies and objects.

Fishing Rod Item Pickup

I modified the fishing rod to pick up items, allowing players to interact with the environment in new ways.

Old Man Room

I implemented the Old Man room, recreating its layout and behavior from the original game.

Level Building

I built the dungeon layout by placing enemies and items to closely match the original level design.

Pull Blocks

I implemented a mechanic that allows the fishing rod to pull blocks. These blocks can be dropped into water and used as walkable platforms.

Designed Custom Level

I designed several custom rooms to showcase the fishing rod. These rooms demonstrate pulling enemies, collecting items, and solving environmental puzzles.
