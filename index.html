The Legend Of Zelda - First Dungeon

By Hamzah Ali

Partners: Joshua Ma, Vera Sivrieva

This project is a recreation of the first dungeon of the classic Nintendo game The Legend of Zelda. The goal was to recreate the feel and structure of the NES game using Unity and C#.

My Contributions

Health, Damage, Knockback, and Hit Flash

I implemented the health and damage system for both the player and enemies. I also added knockback and a visual hit flash effect so combat feels more responsive.

Room Transitions

I created the room transition system that moves the player between rooms using doors, matching the flow of the original Zelda game.

Boomerang

I built the boomerang weapon, including throwing, returning, and interactions with enemies and objects.

Fishing Rod Item Pickup

I modified the fishing rod so it can pick up items, allowing players to interact with the environment in new ways.

Old Man Room

I implemented the Old Man room, recreating the layout and behavior from the original game.

Level Building

I built the game levels by placing enemies and items to match the original Legend of Zelda level design.

Pull Blocks

I implemented a feature with the fishing rod to be able to pull blocks around. These blocks can be dropped into the water and can be walked on.

Designed Custom Level

I designed several rooms and put them together to showcase our fishing rod. I created rooms that display how the fishing rod can pull in enemies, collect items, and pull blocks.
